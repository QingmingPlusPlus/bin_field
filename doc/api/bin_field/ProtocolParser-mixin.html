ProtocolParser

ProtocolParser mixin

A mixin that provides binary protocol parsing capabilities.

This mixin automatically parses binary data according to a list of field definitions. It supports various field types including fixed-length fields, variable-length strings, and null-terminated strings.

Classes that use this mixin must implement fields and content getters.
